Scope defaults (ripple shots)

  • 1× probe (not 10×), short ground at measurement GND.
  • AC coupling, 20 MHz bandwidth limit.
  • Acquire → Peak detect. 20 MHz BW limit knocks down most lamp/EMI above that band; short ground and probing at the DUT reduce pickup. If Pk-Pk jumps when you move the lead or toggle a dimmer, suspect ambient, not the rail.
  • Measure → Pk-Pk on that channel.
  • Line ripple: 1 ms/div (one full 100 Hz cycle @ 50 Hz mains). Switching: 5 µs/div. Open V/div until the trace uses most of the screen height.

C64 Wedge PSU — bench checklist · Sheet 2 of 2 · Steps & measurements

Procedure

A — Safety & setup

  • ☐ Safe bench (e.g. RCD), clear workspace. Probe compensation checked on cal output.
  • ☐ Ripple probe at H3 or same node as e-load (circle one). DC V in the table = e-load display (after cable), not the scope.

B — No load

  • ☐ Power on, electronic load off. Wait ~3 min.
  • ☐ If the load still shows terminal V at ~0 A, write it in the table; else N/A for V.
  • ☐ Line ripple (1 ms/div) → read Pk-Pk → table.
Save PNG — measure-scope-line-ripple-noload.png
  • ☐ Switching ripple (5 µs/div) → Pk-Pk → table.
Save PNG — measure-scope-switching-ripple-noload.png
  • ☐ Heatsink temperature (optional while still cool).

C — 1.5 A

  • ☐ CC load 1.5 A on +5 V. Wait ~3 min (or ~10 min if heatsink was already warm).
  • ☐ E-load V, line Pk-Pk, switching Pk-Pk → table.
Save PNG — measure-scope-line-ripple-load-1p5A.png · measure-scope-switching-ripple-load-1p5A.png

D — Full load

  • ☐ Set 3 A CC load. Wait ≥15 min on heatsink; plateau ≈ <1 °C in 5 min.
  • ☐ E-load V, line Pk-Pk, switching Pk-Pk, T heatsink → table.
Save PNG — measure-scope-line-ripple-load-3A.png and measure-scope-switching-ripple-load-3A.png

E — Done

  • ☐ Copy PNGs off USB. Power off; discharge load safely.
